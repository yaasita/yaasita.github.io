jQuery File Uploadをリバースプロキシ下で使う

デフォルトだとHTTP_X_FORWARDED_HOST等のヘッダは解釈してくれないので、
以下のような設定にしてHOSTヘッダも化かす(apacheの場合)

ProxyPreserveHost On
ProxyPass / http://hoge.example.com/
ProxyPassReverse / http://hoge.example.com/

HTTP_X_FORWARDED_HOSTも解釈して欲しかったら以下のようなパッチを当てる

パッチの当て方

patch -p0 < rproxy.patch
