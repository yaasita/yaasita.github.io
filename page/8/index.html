なんかrsyncでバックアップしようとすると以下のエラーが出るようになった

rsync: read errors mapping "/home/yamasita/Desktop/hogehoge.dat": Input/output error (5)
ERROR: /home/yamasita/Desktop/hogehoge.dat failed verification -- update discarded.

まずbadblocksで調べてみる

ダメそう
とりあえず、fsckをかけてみる

tune2fs -i 30d /dev/sda1

これでとりあえず不良セクタ退避させた

vim-startifyを使ってみる

https://github.com/mhinz/vim-startify

設定

NeoBundle 'mhinz/vim-startify'
    let g:startify_list_order = ['files', 'bookmarks', 'dir']
    let g:startify_files_number = 20
    let g:startify_custom_header =
      \ map(['( ^ o ^ ) / < vim '.v:version]+split(system('date'), '\n'),'"   ".v:val') + ['','']
    let g:startify_bookmarks = split(system('awk "{print \$2}" ~/.NERDTreeBookmarks'),'\n')
